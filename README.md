# Six Seven 67 — landing

Landing page for the **Six Seven 67** app — share links from the app funnel here.
Once GitHub Pages is on: **https://kotensky.github.io/share-six-seven/**

- `index.html` — fully self-contained (inline CSS/JS). Edit + push to update.
- **Enable Pages:** repo → Settings → Pages → *Deploy from a branch* → `main` / `/ (root)`.
- **Make the caps audible:** drop `six.mp3` / `seven.mp3` / `six_seven.mp3` next to `index.html`
  and uncomment the `new Audio(...)` block inside `index.html`.
